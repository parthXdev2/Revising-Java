# Java Practice – Day 4 🚀

This repository contains the programs I built while practicing Java fundamentals on **Day 4 of my learning journey**.

## Concepts Covered
During this practice session, I worked on the following Java concepts:

- Arrays
- User Input in Arrays
- Enhanced Loop (For-Each Loop)
- Java Methods
- Varargs (Variable Arguments)
- Multidimensional Arrays
- Nested Arrays

## Practice Programs
To understand these concepts better, I created small programs and applied them practically instead of only reading theory.

One of the main practice projects I built today was a **Quiz Game in Java**, where questions and options are stored using arrays and the user can answer them through the console.

## Goal
The focus of this project is to strengthen Java fundamentals through hands-on practice and small projects.
